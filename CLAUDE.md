# YunoAna Workspace

This is Ana's personal project workspace. All projects are stored here as subdirectories.

## Structure

Each project lives in its own folder:
```
YunoAna/
├── project-one/
├── project-two/
└── ...
```

## General Guidelines

- Each project may have its own `CLAUDE.md` with project-specific instructions
- When starting work on a project, navigate into its folder first
- Keep projects self-contained — dependencies, configs, and code stay inside their respective folders
